### Descrizione
Il progetto implementa un database per una scuola calcio (o altre attività sportive) individuando le entità principali da manipolare. Dal punto di vista teorico il DB_DesignDocument analizza la struttura del progetto (dallo schema ER iniziale fino alla normalizzazione 3NF).

### Funzionalità
* Inserimento, modifica e rimozione dei tesserati con gestione dei ruoli societari;
* utilizzo di query SQL per avere i principali report;
* app Java che si connette al DB e utilizza le principali query attraverso un menù da linea di comando.
